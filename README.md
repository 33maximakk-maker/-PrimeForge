PrimeForge — Генератор простых чисел на 7 языках
PrimeForge — коллекция из семи независимых реализаций генератора простых чисел с расширенными возможностями. Каждая версия работает на своём языке программирования и предлагает гибкие инструменты для работы с простыми числами.

✨ Уникальные возможности (общие для всех версий)
📊 Генерация простых чисел в заданном диапазоне (до N или от A до B).

⚡ Несколько алгоритмов:

Классический перебор делителей (оптимизированный).

Решето Эратосфена (быстрое для больших N).

🔍 Проверка числа на простоту (в некоторых версиях).

📈 Статистика: количество простых чисел, их сумма, среднее значение, самое большое/маленькое.

💾 Экспорт результата в текстовый файл (в большинстве версий).

⏱️ Замер времени выполнения для сравнения алгоритмов.

🖥️ Интерфейсы:

Командная строка (CLI) с параметрами.

Интерактивный режим (в Python, Java, C#).
